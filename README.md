### 과제 개요

1) 과제 배경
- 최근들어서, 컴퓨터의 성능이 발전하면서 슈팅게임이 3차원 표현을 활용하게 된 시기가 왔다.
컴퓨터의 성능들이 발전해가면서 일반적인 아케이드 게임들이 진화하였으며, 예를 들면 드래곤 플라이트, 제비우스 게임등 다양한 게임들이 발전하게 되었다.
여기에 감명 받아 그림판에서 자기가 직접 그린 그림과 영상으로 피하기 게임을 한다면 어떨까라는 생각을 가지게 되었다.

3) 과제 선택 동기
- 컴퓨터 그래픽스 수업에서 배웠던 펜 만들기, 버튼, 하늘에서 공이 떨어지게 하는 것을 토대로 만드는 것이 좋을 것 같다는 생각을 하게 되었다. 이에 따라서 그냥 그림판을 만들어내는 방식보다는 우리가 직접 그릴 수 있는 그림이 하늘에서 직접 내려오면 어떨가 하는 생각이 들어서 이렇게 과제를 설정하게 되었다.
